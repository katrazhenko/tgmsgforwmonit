# ПОКРОКОВА ІНСТРУКЦІЯ: Налаштування AI Фільтрації

## Крок 1: Реєстрація в OpenAI

### 1.1. Створи акаунт
1. Йди на https://platform.openai.com/signup
2. Зареєструйся (email або Google)
3. Підтверди email

### 1.2. Додай платіжний метод
1. Йди в Settings → Billing
2. Додай кредитну карту
3. Поповни баланс на $5-10 (достатньо на місяць)

### 1.3. Створи API ключ
1. Йди на https://platform.openai.com/api-keys
2. Натисни "Create new secret key"
3. Назва: "Telegram Monitor Bot"
4. Скопіюй ключ (починається з `sk-proj-...`)
5. **ВАЖЛИВО:** Збережи його, більше не покажеться!

## Крок 2: Встановлення бота

### 2.1. Встанови залежності

### 2.2. Перевір встановлення


Якщо виведе "OpenAI OK" - все добре.

## Крок 3: Налаштування бота

### 3.1. Запусти бота
```bash
python main.py
```

Виведе:
```
✅ Запущено бота
🤖 OpenAI доступний - AI фільтрація може бути увімкнена
📢 Надішли /help адміну для довідки
🔄 Фонова пересилка активна
```

### 3.2. Встанови API ключ
Напиши боту в особисті повідомлення:
```
/ai_set_key sk-proj-ТВІЙ_КЛЮЧ_ТУТ
```

Відповідь:
```
✅ OpenAI API ключ збережено
Ключ: sk-proj-Ab...XyZ

Використай /ai_enable щоб увімкнути фільтрацію
```

### 3.3. Увімкни AI
```
/ai_enable
```

Відповідь:
```
✅ AI фільтрація УВІМКНЕНА
Тепер всі повідомлення перевірятимуться GPT
```

## Порівняння: З AI vs Без AI

### БЕЗ AI (тільки мінус-слова)

**Плюси:**
- ✅ Безкоштовно
- ✅ Миттєво
- ✅ Просто

**Мінуси:**
- ❌ Низька точність (~60-70%)
- ❌ Пропускає складну рекламу
- ❌ Треба багато мінус-слів

### З AI (GPT-4o-mini)

**Плюси:**
- ✅ Висока точність (~92%)
- ✅ Розуміє контекст
- ✅ Не треба багато правил

**Мінуси:**
- ❌ Платно (~$1/місяць)
- ❌ Затримка 0.5-2 сек
- ❌ Потребує налаштування

### З AI (GPT-4o)

**Плюси:**
- ✅ Дуже висока точність (~98%)
- ✅ Майже не помиляється

**Мінуси:**
- ❌ Дорожче (~$15/місяць)
- ❌ Затримка 1-3 сек
