⚙️ Systemd-сервіс

Бот запускається як tgmsgforwmonit.service — системний юніт, що стартує автоматично після перезавантаження та перезапускається при падінні.

Unit-файл

/etc/systemd/system/tgmsgforwmonit.service 
[Unit]
Description=Telegram Monitor Bot — tgmsgforwmonit
Documentation=https://your-org.github.io/tgmsgforwmonit
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
User=ubuntu               # замін на свого користувача
Group=ubuntu
WorkingDirectory=/opt/tgmsgforwmonit
EnvironmentFile=/opt/tgmsgforwmonit/.env
ExecStart=/opt/tgmsgforwmonit/.venv/bin/python main.py
Restart=on-failure
RestartSec=10
StandardOutput=journal
StandardError=journal
SyslogIdentifier=tgmsgforwmonit
ExecStopPost=/bin/rm -f /opt/tgmsgforwmonit/monitor.lock

[Install]
WantedBy=multi-user.target
ℹ️ ExecStopPost видаляє monitor.lock при зупинці — захист від ситуації, коли процес впав без коректного завершення.

Усунення проблем

Залишився monitor.lock після краша:

sudo systemctl stop tgmsgforwmonit
rm /opt/tgmsgforwmonit/monitor.lock
sudo systemctl start tgmsgforwmonit

Потрібна повторна авторизація Telegram:

sudo systemctl stop tgmsgforwmonit
rm /opt/tgmsgforwmonit/*.session
cd /opt/tgmsgforwmonit && source .venv/bin/activate && python main.py
# пройти авторизацію → Ctrl+C
sudo systemctl start tgmsgforwmonit

🔎 Логіка фільтрації

Ключові слова — word-boundary матч

Кожен токен ключової фрази перевіряється через регулярний вираз \b...\b. Це означає, що часткові збіги не спрацьовують:

✓ tarjeta → "Mi tarjeta expiró"
Слово цілком — збіг
✗ tarjeta → "tarjetita roja"
Часткове входження — не збіг

Мінус-слова — пряме входження

Перевірка через phrase in text.lower(). Підтримуються мультислівні фрази: "usdt p2p" заблокує "хочу купити usdt p2p сьогодні". Спрацьовує до перевірки keywords — повідомлення відкидається негайно.

/clean_minus — навіщо?

Якщо мінус-слово випадково містить keyword або skip-слово (наприклад, "cita garantizado"), воно може заблокувати цільові повідомлення. Команда /clean_minus видаляє такі токени та прибирає дублікати.

✅ Після /clean_minus: "cita garantizado" → "garantizado". Повідомлення з keyword cita більше не блокується.

🤖 AI-фільтрація

При увімкненому AI кожне повідомлення, що пройшло keyword-фільтр, надсилається в OpenAI GPT. GPT отримує контекст і відповідає одним словом:

TARGET
цільове — пересилається
SPAM
реклама — ігнорується

Що отримує GPT

  • Текст повідомлення
  • Знайдене ключове слово
  • Назву чату
  • Перші 50 keywords та 20 minus_words як контекст
